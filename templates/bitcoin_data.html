Methodology

All figures are estimates using current exchange rates and target block generation times. These figures are updated every 10 minutes, and by combining the current selling price of each digital currency with the number of coins relased each day, we can produce the total amount of USD revenue generated by each coin. For extremely popular coins such as bitcoin and litecoin, the mining power increases so quickly that the average block generation time is slightly faster than the target (for example, each bitcoin block is mined slightly faster than the "target" 10 minute block time) because the computing power of the network grows exponentially. Because of this, the values shown for bitcoin (and to a lesser extent, litecoin), and slightly below the actual values.
